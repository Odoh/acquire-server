### Objects An Acquire game contains of 5 groups of objects: * Bank - contains the draw pile and amounts of stock in the bank. * Board - contains the tiles that are on the board. * Players - contains the id, money, and hand tiles of each player. * Tiles - contains the id and state of each tile in the game. * Hotels - contains the id, state, and prices for each hotel in the game. #### Bank The Bank object contains the size of the draw pile and number of stocks currently in the bank. Example: The Bank contains 108 tiles in its draw pile and 25 stocks of each hotel. { "draw_pile_size": 108, "stocks": { "luxor": 25, "tower": 25, "american": 25, "festival": 25, "worldwide": 25, "continental": 25, "imperial": 25 } } #### Board The Board object contains all the tiles which are currently on the board. Examples: The Board has no tiles on it. { "tiles": [] } The Board has tiles 10-D, 6-E, and 9-I on it. { "tiles": [ "10-D", "6-E", "9-I" ] } #### Players The Players object contains information about a player's money, hand tiles, and stock for each player in the game, indexed by the player's id.
